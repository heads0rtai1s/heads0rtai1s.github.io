Tidy evaluation in R - Complex use cases (feat. ggalluvial)

2019-06-15

In an earlier post I gave a gentle introduction to tidy evaluation in the R tidyverse using simple examples. I covered quoting with enquo and unquoting with !! in brief dplyr and ggplot2 snippets. Today, I aim to build a collection of more complex use cases involving additional tools.

The analogue to enquo for encoding more than one variable is called enquos.

plot_alluvial <- function(df, target, ...){
  
  target <- enquo(target)
  cols <- enquos(...)
  
  df %>% 
    count(!!!cols, !!target) %>% 
    ggplot(aes(axis1 = !!!cols[1], axis2 = clarity, y = n)) +
    geom_alluvium(aes(fill = !!target)) +
    geom_stratum()
  
}

plot_alluvial(diamonds, cut, color, clarity)
